窮人也能架Blog，使用Ghost Buster GitHub Pages

lemotw

A man who interesting road bike, skate board, read, program and the most important thing "spaghetti".

你是否曾想過要架設自己的Blog，卻因沒錢租主機而卻步呢？這篇文將教導你如何在GitHub Pages架設Ghost，正確的說是將Ghost的變靜態並push到GitHub pages上。

建構流程：
1. 安裝必要工具（buster、git）
2. 部屬Ghost
3. 開GitHub Pages專案
4. 使用buster將Ghost轉為靜態
5. 將網站push上GitHub

是不是很簡單呢？真的，很簡單。

(1) 安裝必要工具

#Arch Linux
yaourt -S git ghost  
pip2 install buster

#Debian Ubuntu 用nodejs安裝ghost
#需要mysql node
sudo apt-get install git  
sudo npm install -g ghost-cli@latest  
pip2 install buster

(2) 部屬Ghost
因為Arch的Ghost已經都幫你處理好了，所以下面是Debian/Ubuntu的建制方式

#Arch Linux只要啟動Ghost就好了
systemctl enable ghost

#Debian/Ubuntu
sudo mkdir -p /var/www/ghost

sudo chown ghostexample:ghostexample /var/www/ghost

cd /var/www/ghost  
ghost install

(3)開GitHub Pages專案
不清楚的話可以參考這個網站。

(4)使用buster將Ghost轉為靜態

buster generate --domain=http://127.0.0.1:2368  
buster preview  
#可以開 http://127.0.0.1:9000 看看結果

(5)將網站push上GitHub
